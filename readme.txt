
El objetivo de esta plataforma es ser una herramienta de trading con las WebSockets públicas 
de la plataforma Binance. 

Junto con un host de SMTP se enviarán alertas vía mail según ciertas condiciones de mercado. 

NO será una aplicación que envíe órdenes al mercado de Binance, sinó una herramienta para 
aprovechar del "Carry Trade" ó spread del precio spot y futuro de ciertas criptomonedas.
 
Con este repositorio se habilitará una URL: "HTTPS://JeronimoHoulin.github.io/index.html"
con la cual se podrá acceder a la aplicación sin necesidad de descargarse los archivos
localmente, gracias a Github Pages.
